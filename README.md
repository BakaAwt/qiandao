# 自动签到
可以利用cookie来完成全自动签到，适用于Windows / Linux / Mac / OpenWrt等多平台 <br>

# ！！注意！！
本程序并不会主动将您的账号信息泄漏给第三方 <br>
但并不意味着您的账号处于安全状态  <br>
您在使用本程序时默认已知晓安全性风险。<br>

# 使用
将您的cookie信息复制到 qiandao.ini 内对应位置 <br>
注意，百度仅需要 "BDUSS" 段 <br>
